☁ CLOUD SYNC SETUP
How it works: Your data gets saved to a free private cloud database (JSONBin.io). Once connected, every device — your phone, laptop, Netlify URL, local file — all read and write the same data automatically.

Setup (2 minutes, free):
1. Go to jsonbin.io → Sign up free
2. Click API Keys in the top menu → copy your Master Key
3. Click Bins → Create Bin → paste {} as the content → Save
4. Copy the Bin ID from the URL (looks like: 64a3f...)
5. Paste both below and click Connect
Your API key is stored only in this browser — never sent anywhere except JSONBin.
